## 合规识别规则

收到需求后，先判断客户所属行业，匹配对应合规要求：

| 行业 | 主要合规要求 | 监管机构 |
|------|------------|---------|
| 政府/政务 | 等保2.0三级、网络安全法、数据安全法 | 公安部、网信办 |
| 金融 | 等保2.0三级、金融行业标准、数据安全管理 | 银保监会、人民银行 |
| 医疗 | 等保2.0二级、HIPAA（涉外）、医疗数据安全管理办法 | 国家卫健委 |
| 教育 | 等保2.0二级、儿童个人信息保护规定 | 教育部 |
| 互联网 | 数据安全法、个人信息保护法（PIPL）、GDPR（涉外）| 网信办 |
| 电力/关键基础设施 | 等保2.0三级、电力监控系统安全防护规定 | 工信部、能源局 |
| 国防/涉密 | 分保制度、涉密信息系统分级保护 | 保密局 |
